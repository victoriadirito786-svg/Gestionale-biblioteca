# Gestionale-biblioteca
Si vuole realizzare un applicativo per la gestione di una piccola biblioteca comunale. Il sistema dovra' tenere traccia dei libri disponibili, degli utenti iscritti e delle operazioni di prestito. Ogni libro e' caratterizzato da un titolo, un codice ISBN e un autore. Di ogni autore si vuole memorizzare il nome e la nazionalita'. Un libro puo' essere disponibile oppure in prestito.Gli utenti si iscrivono alla biblioteca fornendo il proprio nome e ricevono un numero di tessera. Ogni utente puo' avere al massimo 3 libri in prestito contemporaneamente.

Quando un utente prende in prestito un libro, viene registrata la data del prestito. Alla restituzione, il libro torna disponibile e viene registrata la data di restituzione. L'applicativo deve consentire di:

aggiungere nuovi libri al catalogo della biblioteca
iscrivere nuovi utenti
effettuare il prestito di un libro a un utente, se il libro e' disponibile e l'utente non ha superato il limite di prestiti
registrare la restituzione di un libro
cercare libri per titolo (anche parziale) o per nome dell'autore
visualizzare tutti i libri attualmente disponibili
visualizzare lo storico dei prestiti di un utente
Il file main deve dimostrare il funzionamento di tutte le operazioni richieste.
